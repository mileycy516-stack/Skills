# AI-Powered Design Intelligence Framework

> **Usage**: Copy sections from here when prompting the agent to generate UI designs.

## ðŸ“‹ The 5 Core Dimensions

### 1. **PATTERN & LAYOUT** (The Skeleton)
*   **SaaS**: Hero + Features + Social Proof + Sticky CTA.
*   **E-commerce (Luxury)**: Full-screen hero slider, grid gallery, zoom.
*   **Fintech**: Split hero (visual + form), live stats, trust badges.
*   **Dashboard**: Bento Grid, modular cards, quick filters.

### 2. **STYLE & AESTHETIC** (The Skin)
*   **Glassmorphism**: `backdrop-filter: blur(10px)`, layered transparency.
*   **Aurora UI**: Vibrant, multi-stop gradients, glowing mesh.
*   **Linear/Vercel**: Dark mode `#0A0A0A`, 1px borders `#333`, minimalist.
*   **Bento Grid**: Structured CSS Grid, card hierarchy, 16-24px gaps.
*   **Neumorphism 2.0**: Soft shadows, comprehensive monochromatic palette.

### 3. **COLOR & THEME** (The Palette)
*   **Trust/Finance**: Navy `#0F172A`, Blue `#0369A1`, Slate Text.
*   **Modern/Creative**: Indigo `#6366F1`, Emerald `#10B981`, White Bg.
*   **Luxury**: Stone `#1C1917`, Gold `#CA8A04`, Cream `#FAFAF9`.
*   **Healthcare**: Cyan `#0891B2`, Teal, Clean White.
*   **Dark Mode**: Bg `#0A0A0A`, Surface `#1A1A1A`, Border `#333`.

### 4. **TYPOGRAPHY** (The Voice)
*   **Tech/SaaS**: Inter (Headers), Roboto/System (Body), JetBrains Mono (Code).
*   **Luxury**: Playfair Display (Headers), Montserrat (Body).
*   **Friendly**: Poppins (Headers), Open Sans (Body).
*   **Editorial**: Merriweather (Headers), Source Sans Pro (Body).

### 5. **ANIMATIONS & INTERACTIONS** (The Soul)
*   **Button Hover**: Scale `1.02`, Lift `translateY(-2px)`, Shadow increase.
*   **Input Focus**: Ring 2-4px, brand color glow, label float.
*   **Cards**: Lift + Shadow, subtle 3D tilt, content reveal.
*   **Transitions**: Staggered list entrance (100ms delay), Modal backdrop fade.

## ðŸš« Anti-Patterns (What to AVOID)
*   **Flash**: No blocking animations or >300ms transitions.
*   **Contrast**: No grey-on-white text (<4.5:1 ratio).
*   **Clutter**: Max 3 primary colors, 2 font families.
*   **Mobile**: No tiny tap targets (<44px).
*   **UX**: No "Mystery Meat" navigation or walls of text.
