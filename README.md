This repository contains agentic skills for the AI assistant, inspired by the `superpowers` repository.

## Structure

- `.agent/skills/`: Contains the defined skills for the AI assistant.
  - `brainstorming/`: Socratic design and requirements gathering.
  - `planning/`: TDD-based implementation planning.
  - `browser-use/`: Automates browser interactions for web testing, form filling, and data extraction using LLMs.
  - `playwright-browser-automation/`: Complete browser automation with Playwright, including dev server detection and responsive design testing.
  - `agent-browser/`: Vercel Labs CLI for deterministic browser automation using refs and AI-optimized snapshots.
  - `browser-workflow-generator/`: Generates comprehensive browser workflow documentation by exploring the app's codebase and researching UX conventions.
  - `react-patterns/`: Modern React patterns and principles, including Hooks, composition, and React 19 features.
  - `web-performance-optimization/`: Optimize Core Web Vitals, bundle size, and rendering speed.
  - `typescript-expert/`: Advanced TypeScript and JavaScript expertise, including type-level programming and performance optimization.
  - `test-driven-development/`: Enforces the TDD cycle (Red-Green-Refactor) for robust feature implementation and bug fixing.
  - `error-handling-expert/`: Master error handling architectures (Exceptions vs Results, Circuit Breakers) across languages.
  - `react-architecture/`: Composition patterns (Compound Components), Hooks, and Next.js best practices.
  - `ui-motion-expert/`: Framer Motion, View Transitions, and micro-interactions for premium feel.
  - `design-system-architect/`: Master design tokens, theming hierarchies, and CVA component architecture.
  - `api-design-principles/`: Best practices for REST/GraphQL resource modeling and contracts.
  - `openapi-spec-generator/`: Create and validate Swagger/OpenAPI v3.1 specifications.
  - `backend-architect/`: Design patterns for microservices, APIs, and distributed systems.
  - `backend-security/`: Secure coding practices, input validation, and auth flows.
  - `platform-engineering/`: Observability (logs/metrics/traces) and performance optimization.
  - `database-specialist/`: Master of SQL, indexing strategies, and PostgreSQL optimization.
  - `prompt-engineering-mastery/`: Expert techniques for Chain-of-Thought, System Prompting, and Hallucination mitigation.
  - `rag-architect/`: Design Retrieval-Augmented Generation systems (Vector DBs, Embeddings, Hybrid Search).
  - `vulnerability-scanner/`: Advanced security auditing, OWASP Top 10, and supply chain security.
  - `security-auditor/`: Expert DevSecOps, compliance (GDPR/SOC2), and cybersecurity.
  - `generating-test-data/`: Create realistic mocks, fixtures, and database seeds (Faker/JSON/SQL).
  - `api-tester/`: Validate REST/GraphQL endpoints with curl, Postman, or Supertest.
  - `database-schema-designer/`: Entity-Relationship modeling, Normalization, and NoSQL patterns.
  - `data-validation/`: Data integrity and type safety with Zod/Joi schemas.
  - `temporal-integration-testing/`: Patterns for testing Temporal workflows (Mocking, Time-skipping).
  - `debugger/`: Debugging specialist for errors and test failures.
  - ...and more.
