# Skills Project

This repository contains agentic skills for Planning and Brainstorming, inspired by the `superpowers` repository.

## Structure

- `.agent/skills/`: Contains the defined skills for the AI assistant.
  - `brainstorming/`: Socratic design and requirements gathering.
  - `planning/`: TDD-based implementation planning.
- `docs/plans/`: Destination for designs and implementation plans generated by these skills.
